# Galaxy

### Crear entorno virtual (Opcional)

```bash
python -m venv venv
```

### Activar entorno virtual (Opcional)

```bash
source venv/bin/activate
```

### Instalar dependencias

```bash
pip install -r requirements.txt
```

### Ejecutar el programa

```bash
python main.py
```

### Crear ejecutable
```bash
 pyinstaller --noconsole --onefile main.py
```
